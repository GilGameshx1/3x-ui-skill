## [2.0.0] — 2024-03-22

### ✨ Новые возможности

- **Полная русификация** — вся документация переведена на русский язык
- **Валидация входных данных** — проверка IP, имени пользователя, домена перед началом установки
- **Индикаторы прогресса** — визуальное отображение прогресса выполнения ([1/22] ✓)
- **SQLite3 поддержка** — автоматическая установка SQLite3 для базы данных 3x-ui
- **Генерация файла-гайда** — автоматическое создание подробного руководства с учётными данными
- **QWEN.md** — конфигурация активации навыка для Qwen Code

### 🛠️ Новые скрипты

- **rollback.sh** — полный откат всех изменений навыка
- **backup-db.sh** — резервное копирование базы данных SQLite3
- **restore-db.sh** — восстановление базы данных из резервной копии
- **test.sh** — тестирование установки и проверка компонентов

### 🔧 Улучшения

- **install.sh** — исправлен путь копирования, добавлена цветовая индикация
- **SKILL.md** — дополнен Step 18A с полным API вызовом для VLESS Reality
- **README.md** — полностью переработан, теперь на русском языке
- **.gitignore** — расширен, игнорирует бэкапы и SSH-ключи

### 📝 Документация

- **QWEN.md** — переведён на русский язык
- **references/vless-tls.md** — переведён на русский язык
- **references/fallback-nginx.md** — переведён на русский язык
- **CONTRIBUTING.md** — руководство для контрибьюторов
- **SECURITY.md** — политика безопасности

### 🐛 Исправления

- Исправлен путь копирования в install.sh (копировал из `skill/` вместо корня)
- Добавлена проверка архитектуры для Reality Scanner
- Улучшена обработка ошибок при потере SSH соединения
- Исправлены команды для генерации ключей X25519

### ⚠️ Критические изменения

- **README.ru.md удалён** — теперь основной README.md на русском языке
- **Структура репозитория изменена** — файлы теперь в корне, а не в папке `skill/`

## [1.0.0] — 2024-03-21

### ✨ Первый релиз

- Базовая адаптация скила из [3x-ui-skill by AndyShaman](https://github.com/AndyShaman/3x-ui-skill)
- Поддержка Qwen Code
- Установка 3x-ui панели
- Усиление безопасности сервера (SSH, UFW, fail2ban, kernel hardening)
- Настройка VLESS Reality и VLESS TLS
- Базовая документация на английском и русском

### 📁 Файлы

- SKILL.md — основной файл навыка
- install.sh — скрипт установки
- README.md — документация (EN)
- README.ru.md — документация (RU)
- references/vless-tls.md — настройка VLESS TLS
- references/fallback-nginx.md — заглушка Nginx
